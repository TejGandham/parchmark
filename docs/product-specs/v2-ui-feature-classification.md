# V2 UI Feature Classification

High-level classification of the features represented in the V2 UI mocks.

## 1. Find Notes

Search, tag filters, sidebar note list, and date groups.

## 2. Read Notes

Reader pane, title, metadata, tags, and markdown content.

## 3. Write Notes

New note, edit mode, and read/edit state switch.

## 4. Manage Notes

Copy markdown, export `.md`, and delete note.

## 5. App Controls

Theme toggle, mobile drawer, user/settings entry, topbar, and sidebar shell.
